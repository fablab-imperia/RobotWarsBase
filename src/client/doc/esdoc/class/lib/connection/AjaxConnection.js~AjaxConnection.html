    AjaxConnection

    Extends:

    Connection → AjaxConnection

    Robot connection implemented with repetitive AJAX PUTs to the robot REST API. In practice, this method is limited to 10-15Hz, but is a lowest common denominator supported by nearly all devices.

    This class is used by the framework and is configured based on options set in the user interface. Direct use by your code is generally not needed.
