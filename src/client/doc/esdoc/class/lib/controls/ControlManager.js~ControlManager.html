    ControlManager

    Manages the display canvas, processing touch events and painting of all user-created controls.

    The class is part of the core framework infrastructure and should not be needed in your code under most circumstances.

    public static convertToPixels(dim: string, value: number | string): number source

    Converts a flexible dimension string pixels for use on the canvas.

    This method supports multiple different formats.

    Params:

    NameTypeAttributeDescription
    dim string
    • nullable: false

    the name of the dimension ('x', 'height', etc.)
    value number | string
    • nullable: false

    raw dimension value, any format

    Return:

    number

    result in pixels

    Example:

    control.dimension = 50     // 50% of the height or width, depending on context
control.dimension = '50%'  // same as above
control.dimension = '20px' // pixels specified directly, only good for one screen size
