    Slider

    Extends:

    Control → Slider

    Slider for uni-dimensional control.

    Sliders can be oriented either vertically or horizontally by setting the Slider#type property. The slider is defined by a reference position, length and radius.

    Vertical Slider

    For vertical sliders, the position is the center of the top half-circle.

    // not actually needed, this is the default mode
mySlider.type = Slider.VERTICAL;
mySlider.position.x = 20;
mySlider.position.y = 20;
mySlider.length = 60;
mySlider.radius = 10;

    Horizonal Slider

    For horizontal sliders, the position is the center of the left half-circle.

    mySlider.type = Slider.HORIZONTAL;
mySlider.position.x = 20;
mySlider.position.y = 20;
mySlider.length = 60;
mySlider.radius = 10;

    Public Members

    public length: number source

    The length of the Slider. This is the distance between the centers of the two half-circles. It will be measured either vertically or horizontally based on the type of slider.

    Example:

    mySlider.length = 60;

    public position: Position source

    The placement of the Slider on the canvas. For a HORIZONTAL Slider, this is the center of the left half-circle, for a VERTICAL Slider, this is the center of top half-circle.

    For dimension values, see ControlManager.convertToPixels.

    Example:

    Set individually
    mySlider.position.x = 10;
mySlider.position.y = 20;
    Set as an Object literal
    mySlider.position = { x: 10, y: 20 };

    public radius: number source

    The radius of the Slider arcs.

    Example:

    mySlider.radius = 10;

    public sticky: boolean source

    Should the slider remember its position when it is not being touched. Normally, this is the behavior you want for something like spinning-weapon power.

    Defaults to true.

    public style: string source

    Style to draw.

    public type: string source

    The type of slider. This must be equal to one of two constants, either Slider.VERTICAL or Slider.HORIZONTAL. These values are just strings of value 'Vertical' and 'Horizontal', respectively.

    Example:

    mySlider.type = Slider.HORIZONTAL;
