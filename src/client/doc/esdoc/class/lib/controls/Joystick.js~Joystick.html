    Joystick

    Extends:

    Control → Joystick

    Joystick for two dimensional X-Y control.

    This is the primary tool for controlling the robot. It functions just like a physical joystick, giving readings in both the X and Y directions.

    Example:

    Setting Joystick parametes in 'setup()'
    var myJoystick = new Joystick('leftStick');
myJoystick.position.x = 40;
myJoystick.position.y = 50;
myJoystick.radius = 15;
    Reading Joystick value in 'loop()'
    var valueX = myJoystick.x;
myMotor.set(valueX);

    Public Members

    public position: Position source

    The placement of the Joystick on the canvas. For dimension values, see ControlManager.convertToPixels.

    Example:

    Set individually
    myJoystick.position.x = 10;
myJoystick.position.y = 20;
    Set as an Object literal
    myJoystick.position = { x: 10, y: 20 };

    public radius: number source

    The radius of the Joystick. Sets the overall size, and therefore sensitivity. Smaller joysticks are more sensitive.

    Example:

    myJoystick.radius = 10;

    public sticky: boolean source

    Should the Joystick reset to zero output when a TouchEvent ends, or continue with the last value. Think of this as "spring return" in a physical Joystick.

    Defaults to false.

    Example:

    myJoystick.sticky = true;

    public style: string source

    Style to draw.

    See:

    • https://developer.mozilla.org/en-US/docs/Web/API/CanvasRenderingContext2D/strokeStyle

    public x: number source

    Output - The horizontal value of the Joystick. This will be a value between -1.0 (full right) and 1.0 (full left). Idle in the center is 0.0.

    Example:

    Get X value
    var myValueX = myJoystick.x;

    public y: number source

    Output - The vertical value of the Joystick. This will be a value between -1.0 (full down) and 1.0 (full up). Idle in the center is 0.0.

    Example:

    Get Y value
    var myValueY = myJoystick.y;
